summary: JCode now makes Catppuccin chat output easier to scan while polishing the sidebar brand, loading mark, and command completion feedback.
---

## Highlights

- **Chat responses are easier to scan:** Catppuccin now has role-based semantic accents for headings, links, inline code roles, and code-block chrome without turning the transcript into a rainbow.
- **The sidebar has clearer structure:** Pinned, Threads, Workspace, and Chats sections now use bold labels with accent icons so the navigation feels less flat.
- **Branding is more distinctive:** The navbar wordmark is larger, with a blood-red `J`, and the splash logo now sits in a padded frame that avoids clipping the mark.
- **Silent commands finish visibly:** Commands that complete without stdout now show completion details with duration and exit code instead of looking unfinished.

## Fixes

- **Loading mark clipping is fixed:** The splash screen logo uses a contained image inside a square-safe frame.
- **Completion output is clearer:** Successful empty-output commands now display a concrete `Completed ... (exit 0)` style summary.
