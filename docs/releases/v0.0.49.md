## Highlights

- **Keybindings are first-class:** Settings now includes a dedicated Keybindings section with search, command groups, source badges, and conflict filters.
- **Shortcut editing is safer:** The recorder blocks unsafe plain letter/number shortcuts and warns before saving over an existing binding.
- **Reset paths are built in:** Individual shortcuts and the full keybindings set can be reset back to defaults from the UI.
- **Advanced users keep control:** The persisted `keybindings.json` remains available as the durable source and advanced escape hatch.

## Fixes

- **Release prep is easier to repeat:** Agent guidance now documents how to prepare future release notes without accidentally publishing early.
