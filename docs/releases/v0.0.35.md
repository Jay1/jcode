## Highlights

- **🗂️ Project import browsing got smarter:** The import palette can now browse nearby paths more directly, helping you find and open the right project location with less guesswork.
- **📊 Provider usage is visible in-context:** The branch toolbar now surfaces provider usage snapshots, making it easier to keep an eye on current usage without leaving your working view.
- **🚀 Desktop startup feels clearer:** JCode now shows a proper splash screen while the desktop backend spins up, so launch feels intentional instead of looking briefly stalled.
- **🎛️ Better provider and theme polish:** Model capability handling, theme editing, and related picker behavior were tightened up so settings feel more consistent and trustworthy.
- **🛠️ Desktop release plumbing is sturdier:** Startup readiness checks, desktop packaging config, and platform entitlements were refined to make desktop builds and app boot more reliable.
