## Highlights

- **🆕 What's new, inline:** Every update now opens a one-time dialog highlighting the latest changes, so you don't have to hunt through a changelog to know what shipped.
- **📚 Release history in Settings:** A full changelog lives under Settings → Release history, grouped by version in a collapsible accordion.
- **Release history stays easy to revisit:** This release remains available in the generated in-app history after GitHub package pages are pruned.
