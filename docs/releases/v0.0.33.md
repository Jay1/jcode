## Highlights

- **📂 Browse local folders right from the composer:** Folder mentions now open a real local directory picker, so you can drill into nearby files and attach the right path without leaving the chat flow.
- **🗂️ Cleaner file and folder mentions:** Mention chips, file trees, and changed-file rows now use a lighter shared icon system that keeps paths easier to scan across the app.
- **🌐 Stronger desktop browser runtime:** The desktop browser path picked up better IPC plumbing, screenshots, clipboard support, and more efficient state syncing for browser-driven tasks.
- **🛟 Smoother startup and provider recovery:** Project hydration, desktop startup, auth visibility, and aborted-turn cleanup were tightened up so sessions recover more predictably after interruptions.
