## Highlights

- **💬 Chats are now available!:** Write without a selected project, or create threads from there.
- **⌨️ New shortcuts:** Quickly open a new chat or jump to your latest project thread with dedicated shortcuts.
- **🧠 Claude 1M context support:** Take full advantage of Claude's 1M-token context window for long conversations and large codebases.
- **📁 Bulk thread actions:** Select multiple threads at once and act on them together.
- **✨ Cleaner reasoning picker order:** The reasoning picker has been reordered to make the most common choices quicker to reach.

## Fixes

- **💻 New polished UI/UX:** A round of visual and interaction polish across the app.
