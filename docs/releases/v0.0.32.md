## Highlights

- **↪︎ Steering messages are clearly marked:** Messages sent with steering now keep a lightweight 'Steering conversation' label above the bubble, even after the app reconciles with the server.
- **🚦 Less aggressive background return checks:** Desktop update checks now wait for a real background return instead of reacting to every tiny blur/focus bounce.
- **🛟 No more stuck checking state:** If the updater never answers, JCode now times out and recovers instead of hanging on a permanent Checking status.
